# Parser Maps Pro

Парсер адресов с Яндекс.Карт с поддержкой SQLite/PostgreSQL.

## Описание

Этот проект предназначен для парсинга и сохранения информации об адресах с Яндекс.Карт. В будущем планируется добавить поддержку 2GIS.

## Технологии

- Python 3.x
- SQLite (по умолчанию)
- PostgreSQL (планируется)
- SQLAlchemy
- Asyncio

## Установка

```bash
pip install -r requirements.txt
```
